Prayerbox Setup Guide

Use this page when you need to connect a box to Wi-Fi for the first time or move it to a new network.

Quick Start

  1. Plug in the Prayerbox and wait while it tries to join Wi-Fi.
  2. If it does not connect, or if your home Wi-Fi changed, hold the physical Prayerbox button for about 5 seconds.
  3. When setup mode starts, join the Wi-Fi network named Prayerbox-Setup-xxxx.
  4. If your phone does not open the setup page automatically, open http://192.168.4.1.
  5. Enter your home Wi-Fi name and password, then tap Save and Restart.
  6. The box will blink three quick times, restart, and reconnect to your home Wi-Fi.
If this is a brand new box, it may still need to be provisioned or claimed in the portal after Wi-Fi is working.

LED Meanings

Slow blink The box is trying to join Wi-Fi.
Fast blink The setup portal is active and ready for you to connect.
Three quick flashes Your Wi-Fi settings were saved and the box is about to restart.
Short failure blink burst Wi-Fi connection failed, and the box is switching into setup mode.
Single brief pulse The box connected successfully to Wi-Fi.

If Setup Does Not Open

  1. Make sure you joined the Prayerbox setup Wi-Fi instead of your normal home Wi-Fi.
  2. Open a browser and go directly to http://192.168.4.1.
  3. If the box still does not appear, unplug it, plug it back in, and hold the button again for 5 seconds.

After Wi-Fi Works

  1. Sign in to Prayerbox.
  2. If the box is new, use the claim code or admin provisioning flow as needed.
  3. Once claimed, use the portal to join a community, set up notifications, and control your light behavior.
